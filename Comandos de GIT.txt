Para trabajar en git hub en local

1.- Instalo en local github
	abro la consola y configuro mi nombre y mi correo
2.- Creo un directorio donde voy a trabajar mi primer repositorio
3.- inicio git hub en ese directorio a traves del command line
	git init
4.- Agrego los primeros archivos al index para que luego los pueda enviar en commit
	git add .
	git add --all
4.- realizo mi primera version:
	git commit -m "Mi Primer commit"


Para subir los cambios a un repositorio remoto (GitHub)
a) Crear un usuario github.com
b) Crear un repositorio en git hub
c) Vincular el repositorio creado en git hub con el repositorio local
	git remote add remoto git@github.com:vcalderon-gb/WilmaSanchez.git
	git remote -v
d) Hacer push de los cambios locales (realizados en mi maquina) al servidor (GitHub)
	git push remote master

En el caso de que el repositorio este creado y quiera hacer una clonacion para tenerlo en local
a) me ubico en el directorio donde voy a crear la carpeta clonada del proyecto y:
	git clone git@github.com:vcalderon-gb/CorpoByte.git
b) luego para subir cambios
	git add index.html
	git commit -m "mensaje"
	git push origin master

Comandos de GIT

Para verificar el estatus de GIT
git status

Para agregar al index un archivo, y que luego pueda ser incluido en el siguiente commit
git add archivo.html

Para preparar los archivo

Para ver el log de los últimos commit realizados
git log

Para ver los los en forma resumida
git log --oneline

Para vincular un repositorio remoto con el que tenemos en local
git remote add remoto git@github.com:vcalderon-gb/WilmaSanchez.git


Para listar todos los remotos a los que tenemos vinculados en local
git remote -v

Para subir los cambios al servidor remoto
git push remote master


Para clonar un repositorio existente en github pero que no lo tenemos en local
git clone git@github.com:vcalderon-gb/CorpoByte.git

Para renombrar un origen
git <nombre actual> rename origin <nombre nuevo>
